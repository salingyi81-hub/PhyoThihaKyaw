📖 What is a Stack?

A Stack is a linear data structure that follows the LIFO (Last-In, First-Out) principle. Think of it like a stack of plates: you can only add a new plate to the top (Push) and only remove the top-most plate (Pop). The Peek operation lets you view the top item without removing it. The Top Index always points to the most recently added item, starting at -1 when empty.
